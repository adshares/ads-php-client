## [0.0.1] - 2018-08-01
### Added
- Support for all block explorer commands:
  - getAccount,
  - getAccounts,
  - getBlock,
  - getBlockIds,
  - getBroadcast,
  - getMe,
  - getMessage,
  - getMessageIds,
  - getTransaction.
- Support for basic transactions:
  - broadcast,
  - sendMany,
  - sendOne.
